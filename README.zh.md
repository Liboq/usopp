基于[Tauri](https://tauri.app/) 复刻 [rubick](https://github.com/rubickCenter/rubick) 项目。

项目初衷用于学习Rust, 代码现阶段凌乱, 后期会进行重构、优化。

如果你有更好的实现, 欢迎提交pr.


### 支持
- 📋 应用程序列表
- 🔎 支持拼音搜索
- 📌 系统最小托盘
- 📁 支持文件夹、文件搜索
